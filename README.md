# data-analysis-practice
파이썬-데이터-실습용

#프로젝트 목적

파이썬을 활용한 csv데이터를 어떻게 전처리하고 가공하여 머신러닝에 사용할지 알아보는 과정중에 실습용. 

#사용 데이터

영화와 평점 데이터를 영화고유번호로 이너조인(merge)한 데이터를 사용

#분석 및 시각화 내용

10년마다 영화 평점에 대한 합계를 히트맵으로 표현

명암에 따른 농도를 지우고 숫자로만 표 형식으로 표현해봄

#사용도구

파이썬

판다스

매트플러립

씨본

#느낀점

변수를 여러개로 두고 그래프를 그릴수 있으면 다양한 결과 분석을 할 수가 있다는걸 깨달았고, 데이터의 의미가 불분명할때도 그래프를 통해서 이 데이터가 무엇을 의미하는지도 알 수 있는
방법이 있다는 것도 알게되었으며, 데이터의 크기나 행 숫자로도 대강의 예측을 할 수 있다는걸 깨달았다.

#향후 계획

이미 존재하겠지만 머신러닝까지 돌리는거 말고 그래프를 그리는거까지만 프로그램화 시켜보면 재밌을거 같다는 생각이 들었고, 재미나이 api를 통한다면 수학을 잘 몰라도 좋은 결과를 낼 수 있을거 같다
